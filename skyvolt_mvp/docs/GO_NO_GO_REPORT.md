## 작성 가이드

이 보고서를 채울 때 RE 가 지켜야 할 원칙:

1. **숫자만 적습니다**. "잘 됐습니다" 같은 표현 금지. "검출률 98.2% (n=605)" 처럼.
2. **그림 1장씩**. 각 EXP 핵심 그래프 1장만. 수십 장 첨부 금지.
3. **나쁜 결과를 숨기지 않습니다**. PARTIAL/BROKEN 도 명확히 적어야 신뢰가 생깁니다.
4. **다운스코프는 정량적**으로. "속도를 좀 낮춥니다" 가 아니라 "1.5→1.2 m/s, 사이클 +0.5s".
5. **Kill criteria 위반 발견 시 즉시 보고서 작성 중단하고 팀 회의** 소집.
